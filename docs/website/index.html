Agentic Workflow

  1. Receives a request from a user.
  2. Analyzes the request and determines that it requires multiple steps to complete.
  3. Generates a Master Plan that outlines the steps needed to complete the request.
  4. Sends the Master Plan to a supervisor agent for review and suggestions.
  5. Sends the suggestions to a bible tool selection agent to select the most appropriate tools.
  6. Sends the selected tools and the Master Plan to an instruction generation agent.
  7. Sends the instructions to an AI assistant for execution.
  8. Monitors the progress and provides additional suggestions or instructions as needed.
  9. Provides a concise summary of the results to the user.
  10. The user receives the final response, which fully resolves their original request.

AI Modes

Chat mode

Provides direct text responses without using tools, ideal for simple queries.

Agent mode

A fully autonomous agent designed to plan, orchestrate tools, and take multiple actions to address user requests.

Partner mode

A semi-automated mode that involves you in the process with review and edit prompts, giving you more control over the AI's actions.
